# fvtt-daggerheart-ko

FoundryVTT용 Foundryborne Daggerheart 시스템의 비공식 한국어 번역입니다.

### 프로젝트 상태

현재 작업이 진행 중입니다. 누구나 기여하실 수 있습니다.

### 설

현재, 다음 링크를 사용하여 수동으로 설치할 수 있습니다:

```
https://raw.githubusercontent.com/psk05020/fvtt-daggerheart-ko/main/module.json
```

### 기여하는 방법

'ko.json' 파일에서 _로 시작하는 키는 번역되지 않은 항목입니다. 해당 내용을 번역한 후에는 맨 앞의 _를 제거해 주세요.

### Update translation

스페인어 번역을 업데이트하려면 다음을 따르세요:

Foundryborne Daggerheart 시스템에서 최신 en.json 언어 파일을 다운로드하세요.

```
npm run update
```

Sync the keys in the `en.json` file with `es.json`.

```
npm run sync
```
